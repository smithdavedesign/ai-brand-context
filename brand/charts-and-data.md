# Charts & Data

## Chart Color Usage
Lead with brand colors. Use accents as highlights only. Ensure legibility between chart sections, type colors, and backgrounds.

**For light backgrounds:**
- Grid lines: 1pt gray
- Text: Dark Blue (unless on dark-colored wedge/section — then use white)

**For dark backgrounds:**
- Grid lines: 1pt white
- Text: White (unless on light-colored wedge/section — then use dark blue)

## Chart Color Rules
- Gradients allowed, but stick to tints or shades of a **single** hue — never mix different hues in the same gradient
- Do NOT use Electric Pink or Electric Teal in charts — difficult on eyes in small, detail-oriented placements
- Do NOT concurrently use gradients and mix different hues from the palette

## Chart-Only Colors (Do Not Use Elsewhere)
- Medium Blue `#475ada`
- Dark Teal `#187e7b`

## Table Styles
- 1pt spacing between columns; alternate colors between rows (no row spacing)
- Highlighted columns may use a thicker border
- Keep data tables minimal — choose 1 font weight and size for row headers, one for data
- Thin rules for row separation (roughly same weight as typography)
- Legal/disclaimers: small but legible, below chart, left-aligned
